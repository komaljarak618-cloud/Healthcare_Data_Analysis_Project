# Healthcare Data Analysis Project

## Overview
This project focuses on healthcare data cleaning and preprocessing using Python and Pandas.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Project Workflow
1. Load healthcare dataset
2. Check missing values
3. Handle duplicate records
4. Clean and preprocess data
5. Export cleaned dataset
6. Perform exploratory data analysis

## Dataset
Healthcare dataset used for data cleaning and analysis.
